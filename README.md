# EkiTili - Изучение казахского языка

Приложение для изучения казахского языка с системой карточек и уроками.

> 🚀 **Новичок?** Начните с [QUICK_START.md](QUICK_START.md) для быстрого запуска или [GUIDE.md](GUIDE.md) для подробной инструкции.

## Модули

### Конфигурация (`src/config/`)
- **constants.js**: Константы приложения (интервалы SRS и т.д.)

### Данные (`src/data/`)
- **lessons.js**: Данные уроков и прогресс
- **flashcards.js**: Данные карточек и пользовательские карточки
- **leaderboard.js**: Данные лидерборда
- **user.js**: Данные пользователя (профиль, стрик, активность)

### Сервисы (`src/services/`)
- **storage.js**: Работа с localStorage для сохранения прогресса
- **srs.js**: Логика системы интервального повторения (SRS)
- **stats.js**: Вычисление статистики (процент курса, активность)

### Модули (`src/modules/`)
Каждый модуль разделен на логику и рендеринг:
- **lessons/**: Управление уроками
- **flashcards/**: Управление карточками
- **leaderboard/**: Лидерборд
- **profile/**: Профиль и статистика

### Утилиты (`src/utils/`)
- **dom.js**: Вспомогательные функции для работы с DOM
- **date.js**: Функции для работы с датами
- **charts.js**: Функции для рисования графиков на canvas

### Навигация (`src/navigation/`)
- **router.js**: Управление переключением между вкладками

## Использование

**ВАЖНО:** ES6 модули не работают при открытии файла напрямую через `file://` протокол из-за политики безопасности браузера (CORS). Необходимо запустить локальный HTTP сервер.

### Быстрый запуск

Скрипты запуска автоматически запускают:
- **Веб-сервер** (порт 8000) - для фронтенда
- **Бэкенд API сервер** (порт 3000) - для подключения к Oracle Database

**Windows:**
```bash
start-server.bat
```

**Linux/Mac:**
```bash
chmod +x start-server.sh
./start-server.sh
```

Затем откройте браузер и перейдите по адресу: `http://localhost:8000`

### Первый запуск

При первом запуске скрипт автоматически установит зависимости Node.js. Убедитесь, что:
1. Установлен **Node.js** (https://nodejs.org/)
2. Установлен **Oracle Instant Client** (см. [SETUP_ORACLE.md](SETUP_ORACLE.md))
3. Настроены параметры подключения к Oracle в `server.js`

### Настройка подключения к Oracle

Откройте файл `server.js` и измените параметры подключения:

```javascript
const dbConfig = {
    user: 'SYSTEM',                    // Ваш пользователь Oracle
    password: 'ваш_пароль',            // Ваш пароль
    connectString: 'localhost:1521/XE' // Ваш connection string
};
```

Подробные инструкции по настройке Oracle см. в [SETUP_ORACLE.md](SETUP_ORACLE.md)

### Альтернативные способы

**Python:**
```bash
python -m http.server 8000
# или
python3 -m http.server 8000
```

**Node.js (http-server):**
```bash
npx http-server -p 8000 -c-1
```

**Node.js (live-server):**
```bash
npx live-server --port=8000
```

**VS Code:**
- Установите расширение "Live Server"
- Правый клик на `index.html` → "Open with Live Server"
